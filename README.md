# portfolio_website
UX portfolio website showcasing my design workflow, projects, and detailed case studies.

**Project Overview

This project focuses on the design and development of a modern website based on a gym and fitness theme. The goal of the project was to explore a complete UX design workflow, from research and inspiration through to prototyping, development, and usability testing. The website was designed to prioritise clear visual hierarchy, strong layout structure, and improved user experience while maintaining a visually engaging interface suited to a fitness environment.

The project followed a structured UX process which included design inspiration research, persona development, wireframing, prototyping, development, and accessibility testing.

**Design Goals

The project aimed to achieve the following objectives:
- Create a clear and engaging homepage layout
- Improve visual hierarchy and content structure
- Ensure navigation is simple and intuitive
- Create a responsive design that works across devices
- Maintain consistent typography and spacing
- Integrate imagery effectively within the layout
- Ensure accessibility standards are met

**UX Design Process
Research & Inspiration

Several existing websites and design examples were analysed to understand effective layout patterns, visual hierarchy, and case study presentation techniques. These references helped guide the design direction and provided inspiration for elements such as hero sections, content structure, and visual styling.

**User Persona
A recruiter persona was created to represent a typical user reviewing a UX portfolio. This helped identify key goals, behaviours, and pain points when evaluating design work, ensuring the project presentation is easy to understand.

**Sketching & Ideation
Early design ideas were explored through quick sketches to experiment with different homepage structures and content layouts.

**Low Fidelity Wireframes
Low fidelity wireframes were created to define the structural layout of the website. These focused on functionality and navigation before introducing visual styling.

**High Fidelity Prototype
A high fidelity prototype was then developed to refine the visual design. This included improved typography, spacing, imagery placement, and overall layout consistency. Multiple iterations were made during this stage to refine the design and improve usability.

**Development
The final design was implemented using modern web technologies including:
- HTML5
- CSS3
- JavaScript

The website was built using semantic HTML and responsive design principles to ensure compatibility across different screen sizes. Styling and layout were structured to maintain consistent spacing, typography, and visual hierarchy throughout the interface.

**Testing & Optimisation
The completed website was tested to ensure it met performance, accessibility, and usability standards.

**Accessibility Testing
Colour contrast testing was conducted using a Contrast Checker to ensure text and background colours meet WCAG AA and AAA accessibility standards.

**Lighthouse Testing
The site was tested using Google Lighthouse to evaluate:
- Performance
- Accessibility
- Best Practices
- SEO

Issues identified during testing were addressed through improvements such as semantic structure adjustments, image optimisation, and layout refinements.

**Outcome

The final website demonstrates a complete UX workflow from research through to implementation. The finished design provides a clear layout structure, improved visual hierarchy, and a responsive interface that enhances usability across devices. Through iterative prototyping and testing, the project ensures accessibility standards are met while delivering a visually engaging and user-friendly experience.
